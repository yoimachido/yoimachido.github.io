このページは、このサイトの実装についてのメモです。

## このサイトは何で動いているか

GitHub Pages の上で動いている静的サイトです。データベースはなく、サーバー側の処理もありません。テキストファイルを直接書いて、Git で管理しています。

## なぜ静的サイトにしたか

長く続けやすい形にしたかったからです。依存するものが増えるほど、何かが変わったときに崩れやすくなります。少ないもので動かしておいたほうが、長持ちしやすいと思っています。

CMS も使っていません。テキストファイルを直接編集し、変更は Git で追います。自分で管理したかった、ということです。

## いまの構成

コンテンツは三種類に分けています。

- **works** — 短編フィクション
- **notes** — 覚書・断片
- **labs** — このサイト自体の記録（このページもここに入ります）

混ぜると見えにくくなるので、分けました。整理は読者のためだけでなく、自分の棚を把握するためにもあります。

## まだできていないこと

- タグ・シリーズ別の一覧ページ
- OGP 画像の設定

コンテンツが増えてきたタイミングで、順番に手を入れていく予定です。

## これから

何かを変えたとき、なぜそうしたかを残せるよう、labs に記録を積んでいきます。ひとまず今回は、最初の実装メモとしてここまでです。
